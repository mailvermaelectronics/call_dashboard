# Call Dashboard PRO+

Premium iPhone-style call dashboard with charts, filters, and auto-refresh.

## How It Works:
MacroDroid → HTTP PUT → GitHub API → data.json update → Dashboard loads it.

## Features:
- Clean iPhone-style UI
- Live table
- Search + Filters
- Call type tabs
- Dark/Light mode
- Auto refresh every 10s
- Smooth animations
- CSV Export
